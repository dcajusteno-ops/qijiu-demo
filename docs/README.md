# Comfy Manager 使用文档

当前稳定版本：`v2.0.1`

## 项目简介

Comfy Manager（灵动图库）是一个面向 **ComfyUI 输出目录** 的桌面图片管理器，使用 **Wails v2 + Go + Vue 3** 构建。

它解决的核心问题是：

- ComfyUI 出图后难以快速回看
- 图片、模型、LoRA、Prompt 信息分散
- output 目录不固定时，传统管理工具难以适配
- 需要更轻量地做收藏、打标、筛选与归档

## v2.0.1 功能概览

### 核心能力

- 绑定任意 ComfyUI `output` 目录
- 默认目录 / 日期归档目录 / 自定义目录并行浏览
- 日期产出工作台
- 模型 / LoRA 高级筛选
- 搜索文件名、Prompt、模型、LoRA、标签、笔记
- 收藏夹与收藏分组
- 自动规则引擎
- PNG 元数据查看与复制
- 批量模式

### 本次版本新增重点

- 更新软件内置“使用文档”页面，使其与外部文档保持一致
- 修复内置文档页中的中文乱码和旧版本残留
- 新增设置中心，集中管理常用系统功能
- 新增工具菜单配置：支持顺序调整、显示/隐藏
- 日期产出工作台支持日期范围筛选
- 修复首页“查看全部”跳转空页问题
- 修复多处中文乱码与目录名称异常显示

## 常用入口说明

### 工作台总览

用于查看：

- 最新作品
- 今日新增
- 图片总数
- 存储占用
- 最近更新入口

### 日期产出工作台

适合围绕日期回看近期出图：

- 今天 / 昨天 / 最近 7 天 / 本月
- 自定义日期范围
- 叠加模型筛选
- 叠加 LoRA 筛选
- 点击最近活跃日期后，一键跳回图库

### 图库页

适合进行细看和批量整理：

- 搜索
- 模型 / LoRA 筛选
- 标签筛选
- 收藏筛选
- 排序与分页
- Lightbox 详情查看

### 自动规则引擎

适合做自动化整理：

- 按模型自动打标
- 按 LoRA 自动打标
- 按目录归类
- 自动收藏或补充信息

## v2.0.1 使用建议

1. 首次启动时，先绑定真实的 ComfyUI `output` 目录。  
2. 如果你有常用子目录，可以在“工具菜单 -> 设置 -> 文件夹维护 / 自定义目录”里整理侧边栏。  
3. 回看近期产出时，优先使用“日期产出工作台”。  
4. 需要做统一归类、打标和后处理时，使用“自动规则引擎”。  
5. 如果工具菜单太长，可以在设置中心调整顺序和隐藏不常用项。
